# XF2-Swedish
This is a complete translation of XenForo 2.x.x to Swedish.

# About me
I'm Jill Karlsson and the site owner of the swedish [Thaisnack Forum](https://thaisnack.se/). You may contact me on using the [Contact Form](https://thaisnack.se/misc/contact/).

#### Prerequisits
* [xenForo](https://xenforo.com) version **2.0** or later

## Installation
To import a language in ACP, choose the Import a Language option in the Appearance tab. You have the option to create a new language or overwrite an existing one. If you overwrite an existing language, any customizations you have made will be overwritten by the language youâ€™re importing.
